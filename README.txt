# Pressure_sensor_3_channel
Микроконтроллер считывает с 3-х датчиков давления показания. 
Внури есть фильтрация по среднему значению.
Затем по ком порту передает на ПК, здесь программа на PyQt6 ловит сигнал преобразовывает его в давление и аналоговое напряжение.
В программе присутствует калибровка нуля, проверка подключения.
